## Key Bindings
### Autocompletion
`<Enter>` (i) - confirm autocompletion

### Git
`<Space>lg` (n) - open lazygit menu (`:LazyGit`)\

### NeoTree
`<Ctrl>n` (n) - reveal filesystem (`:Neotree filesystem reveal left`)

### Telescope
`<Ctrl>p` (n) - find files\
`<Space>fg` (n) - live grep

### ToggleTerm
`<Ctrl>t` (n) - open/close terminal

### Other
`<Ctrl><Backspace>` (i) - delete word
